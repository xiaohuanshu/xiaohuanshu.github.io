第一个程序 helloworld

Date 六 17 十二月 2016 Tags c语言 / helloc / 教程
  • helloworld是IT界开始的象征，你学习任何一门计算机编程语言都将是输出一行helloworld，代表你真正的走入了这个世界。
  • 比如C++语言： cout<<”helloworld!”;
  • 比如java语言： System.out.println("helloworld");
  • 比如basic语言： print “helloworld”
  • C语言实现输出helloworld的完整代码：
#include <stdio.h> 
int main(void){
    printf("helloworld");     
    return 0; 
}
  • 短短的5行代码，将是你走入编程世界的第一步，我们先不着急理解每句话什么含义，先运行一下这段代码体会一下成果吧。

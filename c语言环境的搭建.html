c语言环境的搭建

Date 六 17 十二月 2016 Tags c语言 / helloc / 教程
  • 等等，进入编程世界的大门之前，还有很关键的一步没有做，那就是请C语言翻译官，也就是编译器和开发环境入住我们的电脑，笔者在这里推荐广大的C语言新手安装Dev-Cpp这款软件，这款软件使用简单并且完全免费，大家可在网上搜索并下载。
  • 下载并安装好后界面：
  • 好了，现在C语言翻译官已经入住我们的电脑了，那么我们尝试去与这个翻译官沟通吧，尝试在这个界面随便操作一下试试，看看能不能将前面helloworld程序执行。笔者在这里打保证，无论你怎么随意的操作，电脑都不可能坏的。
  • 相信你们已经迫不及待的想要运行一下刚才的程序了，Dev-Cpp和我们的记事本或者word一样，每次想编写一个文件都需要新建、保存。C语言同样是要存在文件中的，所以我们的第一步：(菜单)文件->新建->源代码。
  • 然后是保存，（菜单）文件->保存。切记保存的时候保存类型要选择C source files(*.c)，因为我们写的是C语言代码。
  • 随后，我们就将刚才helloworld中的代码敲入中间的编辑区域吧，就像word一样。 ￼￼*
  • 最激动人心的时刻到来了，翻译官已经听到了我们的命令，现在要让翻译官翻译我们的程序命令让电脑执行！果断的按下键盘上的F11按键吧！
  • 这个黑乎乎的窗口，就是电脑执行的结果，helloworld已经静静的显示在了上面，恭喜你，你已经迈入了编程世界的第一步，如果你在这一步并没有出现这个黑乎乎的窗口，取而代之的是编辑区域内的一条条红道，说明你在走入编程世界的时候并没有细心，仔细对比一下书中的代码看看那里抄错了吧。

  • 第一个很简单，聪明的你一定知道，只要将代码中的”helloworld”换成”iloveyou”就可以了。但是第二个程序，你会发现代码写成这样：
#include <stdio.h> 
int main(void){
    printf("我爱你");   
    printf("中国");   
    return 0; 
}
  • 但是执行的结果却是这样的：
  • 为什么没有换行呢？这还是因为电脑太傻了，它只会将你给的内容直接输出，但是并不会换行，如果需要换行的话你需要告诉电脑在哪里换行，尝试用一下换行符“\n”吧。
#include <stdio.h> 
int main(void){
    printf("我爱你");   
    printf("\n");   
    printf("中国");   
    return 0; 
}
  • 或者这样：
#include <stdio.h> 
int main(void){
    printf("我爱你\n中国");   
    return 0; 
}
